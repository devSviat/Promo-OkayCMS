# Модуль Promo: акції та знижки для OkayCMS

Модуль дозволяє створювати маркетингові акції в магазині: відсоткові та фіксовані знижки, подарунки, механіку `1+1=3` і безкоштовну доставку. Акції показуються на сторінках товарів, у каталозі, в кошику та на окремих лендингах акцій.

## Можливості

### Типи акцій

- **Відсоткова знижка**
- **Фіксована знижка**
- **Подарунок**
- **1+1=3 (bundle)** — третя одиниця безкоштовно
- **Безкоштовна доставка**

### Гнучка область дії акції

- Включення за:
  - категоріями
  - брендами
  - конкретними товарами
  - значеннями властивостей (feature values)
- Підтримка **виключень** (exclude), щоб прибрати частину товарів з акції.
- Мінімальна сума замовлення для спрацювання акції.
- Пріоритет акцій (яка застосовується раніше).

### Відображення у вітрині

- Бейдж акції на фото товару.
- Блок акції у картці товару.
- Таймер до завершення акції (або до старту, якщо акція майбутня).
- Окрема сторінка списку акцій: `/promo`.
- Окремий лендинг конкретної акції: `/promo/{url}`.

### Подарунки та кошик

- Для акцій типу «Подарунок» можна вказати список подарункових товарів.
- Подарунки відображаються в кошику та замовленні як безкоштовні позиції.

### Робота з фідами (опційно)

- Передача промо-даних у фіди, якщо встановлені модулі:
  - `OkayCMS/Feeds`
  - `OkayCMS/GoogleMerchant`

## Вимоги

- OkayCMS v4.5.2
- (Опційно) `OkayCMS/Feeds` та/або `OkayCMS/GoogleMerchant` — якщо хочете передавати акції у фіди

## Встановлення

### Варіант 1: Скачування з GitHub

1. Скачайте останню версію модуля з [останнього релізу на GitHub](https://github.com/devSviat/Promo-OkayCMS/releases/latest).
2. Розпакуйте архів на сервері в каталог: `{OkayCMS_root}/Okay/Modules/`

### Варіант 2: Встановлення через Git

1. Виконайте команду в директорії `{OkayCMS_root}/Okay/Modules/Sviat`:
   ```bash
   git clone https://github.com/devSviat/Promo-OkayCMS.git Promo
   ```

### Результат встановлення

Шлях до модуля має бути таким:
```bash
{OkayCMS_root}/Okay/Modules/Sviat/Promo/
```

Після встановлення перейдіть в адмін-панель → Модулі → Знайдіть «Promo: акції та знижки» → Натисніть «Встановити».

## Налаштування та запуск акції

1. Перейдіть в адмін-панель → Каталог → Акції.
2. Натисніть «Додати акцію».
3. Заповніть основні поля:
   - назва
   - URL
   - тип акції
   - пріоритет
   - мінімальна сума замовлення (за потреби)
4. Налаштуйте область дії (категорії/бренди/товари/властивості) і, за потреби, виключення.
5. Для типу «Подарунок» додайте подарункові товари.
6. За потреби увімкніть період дії (дата старту/завершення).
7. Додайте зображення (банер, мобільне, бейдж), якщо потрібно.
8. Збережіть акцію.

## Використання

- Покупець бачить акційний бейдж і блок акції на товарі.
- У кошику акція застосовується автоматично, якщо умови виконані.
- Для майбутніх акцій відображається статус «незабаром» і таймер до початку.

## Важливо

- Для акції типу «Подарунок» потрібно обрати хоча б один подарунок.
- Для будь-якої акції потрібно задати хоча б один об’єкт включення (категорія/бренд/товар/властивість).
- Якщо одночасно діє кілька акцій, порядок застосування залежить від пріоритету.
- URL акції має бути унікальним.
